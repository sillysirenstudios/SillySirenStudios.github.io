Capabilities

Everything you need,
nothing you don't

Built natively in SwiftUI with Citadel and SwiftTerm — not a web view.

⚡

Native SSH Terminal

Full terminal emulation via SwiftTerm. Supports xterm-256color, bold, italics, and proper escape sequences.

🔑

Flexible Authentication

Password, ed25519, ECDSA (P-256/384/521), and RSA key auth. TOFU host key pinning out of the box.

🗂

SFTP File Browser

Navigate directories, edit text files, and detect binaries — all over a dedicated SFTP connection. Pro

🪟

tmux Integration

Browse sessions and windows, create or kill sessions, and drill into windows without leaving the app. Pro

🔀

Git Diff Viewer

Colored unified diffs, staged/unstaged toggling, commit log with branch display — all over SSH. Pro

🖥

Multi-Server Sessions

Connect to multiple servers simultaneously. Sessions persist across sidebar navigation for the app lifetime. Pro

🎨

Fully Customizable

Font family, font size, color schemes, and system/light/dark appearance — tuned to your taste.

🔒

Zero Telemetry

No analytics, no crash SDKs, no ad frameworks. Your data goes directly from your iPad to your server.
