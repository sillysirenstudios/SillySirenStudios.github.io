How it works

What is TestFlight?

TestFlight is Apple's official beta testing platform — safe, free, and separate from the App Store.

1

Get an invite

Send us an email and we'll send you a TestFlight invitation link directly to your inbox.

2

Install TestFlight

If you don't already have it, download the free TestFlight app from the App Store.

3

Install the beta

Open the invite link, tap Accept, and install Silly Siren Terminal. You'll receive updates automatically as we ship new builds.

4

Share your feedback

Found something? You can send feedback directly from TestFlight, or just reply to the invite email. We read everything.

What to test

Where to focus

Here's what we'd love eyes on. Pro features are fully unlocked in the beta — no purchase needed.

Getting started

  • Add a server and connect via SSH (password or key-based auth)
  • Test on your local network or a remote server

Unlocking Pro

  • Open Settings (gear icon in the server list)
  • Scroll to the bottom → Developer → flip "Pro Unlocked" on
  • This gives you full access to all Pro features without any payment

Terminal

  • Connect, type commands, resize the terminal
  • Try different color themes

Files (Pro)

  • SFTP browser — navigate directories
  • Open and edit text files over SFTP

tmux (Pro)

  • List sessions, create and kill sessions
  • Drill into windows without leaving the app

Git (Pro)

  • View changed files and colored diffs
  • Browse the commit log

Settings

  • Font family, size, and color themes
  • Import a custom theme (.json or .itermcolors) with Pro unlocked

Multi-server (Pro)

  • Connect to multiple servers simultaneously
  • Swipe to delete a server and confirm the delete dialog
Known limitations

A few things to keep in mind

We're aware of these — no need to report them, but good to know before you dive in.

  • RSA keys may be rejected by modern servers — ed25519 works best
  • Passphrase-protected keys are not yet supported
